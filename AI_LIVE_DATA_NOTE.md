# AI Live Data Note

If you are an AI working in this repo:

- The live Synology Monitor history is not in these files.
- The live findings, conversations, recommendations, and remediation results are in Supabase.
- In this environment, use the configured Supabase MCP first when the user asks about current or past monitor state.

Examples:
- what problems the monitor found
- what the AI already recommended
- what actions were proposed or executed
- what happened in prior issue conversations
